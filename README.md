🚦 Smart Traffic Violation Detection System

📌 Project Overview
The Smart Traffic Violation Detection System is an AI-powered application that automatically detects traffic rule violations using Computer Vision and Deep Learning. The system analyzes traffic images or videos and identifies vehicle that violate traffic rules.

🎯 Objectives
🚗 Detect vehicle automatically
🛣️ Detect lane violations
🚦 Detect signal jumping
🪖 Detect helmet violations
⚡ Detect overspeeding vehicles
🤖 Reduce manual traffic monitoring

▶️ How to Run
1. Install all required libraries.
2. Download the YOLOv8 model.
3. Add a traffic image or video.
4. Run the project using: python app.py

📊 Expected Output
1. Detect cars, bikes, buses, trucks and people
2. Highlight detected objects with bounding boxes
3. Identify traffic rule violations
4. Display AI detection results
